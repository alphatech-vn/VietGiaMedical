# Privacy Summary

VietGia Medical ERP is an internal business application.

High-level privacy expectations:

- access should be role-based,
- project, customer, personnel, and financial data are sensitive,
- operational visibility depends on internal permissions,
- production data handling is not documented publicly in this repository.

This repository does not publish:

- source code,
- internal credentials,
- project records,
- personnel records,
- financial or tender data exports.
