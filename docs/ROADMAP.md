# Public Roadmap

This is a lightweight public roadmap for VietGia Medical ERP.

## Current product shape

- dashboard,
- projects and project detail,
- tasks,
- reports,
- analytics,
- personnel,
- settings,
- system administration surfaces.

## Likely next-value areas

- deeper workflow automation,
- stronger cross-role task coordination,
- richer tender-stage reporting,
- more mature governance and operational analytics.

## Scope note

This roadmap is directional only. It is not a release commitment.
